📖 Introduction & Core Concepts

Computers use different types of memory to do different jobs. Some types of memory are incredibly fast but small, while others are huge but quite slow.

The golden rule to remember is: The closer memory is to the computer's brain (the CPU), the faster it works, but the less space it has.

💡 Key Definition: Volatile Memory

Volatile means temporary. If memory is volatile, it clears out and deletes everything the exact second the computer loses power or gets turned off.

💡 Key Definition: Non-Volatile Memory

Non-Volatile means permanent. This type of memory keeps your files and data safely saved even when the computer is completely unplugged.

🔍 The Four Layers of Memory

Scroll through the boxes below to see how memory changes from top to bottom.

🥇 1. CPU Registers

These are tiny storage slots built right inside the CPU core itself. They hold only one calculation at a time, but they have absolutely zero lag.

Temporary (Volatile)
🥈 2. Cache Memory

Sits right next to the CPU. It saves copies of data you use frequently so the computer's brain doesn't have to keep fetching them from the slower main memory.

Temporary (Volatile)
🥉 3. Main Memory (RAM)

This holds the active applications and browser tabs you are using right now. It has a lot of room, but it is noticeably slower than Cache.

Temporary (Volatile)
💾 4. Secondary Storage (SSD/HDD)

This is your actual hard drive where all your games, photos, and operating system files live long-term. It is slow, but massive.

Permanent (Non-Volatile)
