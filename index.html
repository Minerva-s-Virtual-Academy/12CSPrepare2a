Q1: Why does a standard computer system require Secondary Storage if Main Memory (RAM) is significantly faster?
Expected Criteria: Main Memory (RAM) is fundamentally volatile, meaning it loses its state entirely when electrical current drops out. Secondary storage is strictly non-volatile, meaning it is required to permanently preserve application assets, files, and the OS system kernel long-term.
Q2: Explain how the use of high-speed Cache memory prevents the 'Von Neumann Bottleneck'.
Expected Criteria: The CPU runs much faster than the data bus lines reading from main memory. If the CPU had to fetch every instruction directly from RAM, it would spend most of its time idling (stalling). Cache stores frequently used instructions close to the core using fast SRAM, keeping the execution loop moving.
